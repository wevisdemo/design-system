We provide CSS classes for font family, weight and size

Font Family

wv-[family]

  • Kondolar Thai : Heading
  • IBM Plex Sans Thai : Heading or body
  • [DEPRECATED] Anuphan : Heading or body
  • Bai Jamjuri : Body
<span class="wv-kondolar">Kondolar Thai</span>
<span class="wv-ibmplex">IBM Plex Sans Thai</span>
<span class="wv-anuphan">[DEPRECATED] Anuphan</span>
<span class="wv-baijamjuri">Baijamjuri</span>

Font Weight

wv-[weight]

Semibold

  • IBM Plex Sans Thai Semi-Bold 
<span class="wv-ibmplex wv-semibold">IBM Plex Sans Thai Semi-Bold</span>

Bold

  • Kondolar Thai Bold
  • IBM Plex Sans Thai Bold
  • Bai Jamjuri Bold 
<span class="wv-kondolar wv-bold">Kondolar Thai Bold</span>
<span class="wv-ibmplex wv-bold">IBM Plex Sans Thai Bold</span>
<span class="wv-baijamjuri wv-bold">Bai Jamjuri Bold</span>

Black

  • Kondolar Thai Black 
<span class="wv-kondolar wv-black">Kondolar Thai Black</span>

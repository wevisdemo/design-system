<div class="not-prose component-preview">

</div>

```jsx

import WvNavButton from '@wevisdemo/ui/{react,vue2,svelte}/nav-button';

<WvNavDropdownItem>Topic 1</WvNavDropdownItem>
```

| Name | Type    | Default |

## Slot / Children

Item content

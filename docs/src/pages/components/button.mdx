## Props

| Name    | Type                               | Default     |
| ------- | ---------------------------------- | ----------- |
| color   | `'blue'` \| `'black'` \| `'white'` | `'blue'`    |
| small   | boolean                            | `false`     |
| onClick | () => void \| undefined            | `undefined` |
